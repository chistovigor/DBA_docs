Простое

emca -config dbcontrol db

если не работает:

emca -deconfig dbcontrol db

emca -config dbcontrol db

Пересоздание репозитория

emca -deconfig dbcontrol db

(!!! БД переходит во время удаления репозитория в ограниченный режим)

или

emca -deconfig dbcontrol db -repos drop

emca -config dbcontrol db -repos recreate

создание:

emca -config dbcontrol db -repos create

emca -config dbcontrol db

Нужны файлы в каталоге (они одинаковы для одинаковых версий Oracle)

/usr/oracle/app/product/11.2.0/dbhome_1/oc4j/j2ee/OC4J_DBConsole/

Запуск консоли

/usr/oracle/app/product/11.2.0/dbhome_1/bin/emctl start dbconsole

остановка 

/usr/oracle/app/product/11.2.0/dbhome_1/bin/emctl stop dbconsole
